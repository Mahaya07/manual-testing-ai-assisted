# AI-Assisted Test Case Design

## Prompt Used
"Generate positive, negative, and edge test cases for a login feature of a web application."

## AI-Generated Suggestions
- Invalid username format
- Password length validation
- SQL injection attempt
- Login with special characters
- Account lock after multiple failures

## QA Review Notes
- Selected relevant scenarios applicable to manual testing
- Ignored security cases beyond test scope
- Converted valid scenarios into executable test cases

AI was used to expand thinking, while final decisions were made manually.
