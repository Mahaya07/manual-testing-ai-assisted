
# Manual Testing Project (AI-Assisted)

## Project Overview
This project demonstrates manual testing fundamentals along with the use of AI tools to enhance test case design and bug reporting quality.

## Scope of Testing
- Functional testing
- Positive and negative test scenarios
- Basic UI validation

## AI-Assisted Approach
AI tools were used to:
- Generate additional test scenarios
- Identify edge cases often missed in manual testing
- Improve clarity and completeness of bug reports

All AI-generated suggestions were reviewed and validated manually.

## Project Structure
- test-cases/ – Manual test cases
- bug-reports/ – Sample bug reports
- ai-prompts/ – AI prompts and review notes
- screenshots/ – Evidence and visuals

## Tools Used
- Manual Testing Techniques
- AI-assisted analysis
- GitHub documentation
